## 💡 Solution — EnergyGrid

A **peer-to-peer energy trading platform** with three core layers working together:

```
┌─────────────────┐    ┌──────────────────┐    ┌─────────────────────┐
│   AI Layer      │    │  Trading Layer   │    │    Trust Layer      │
│─────────────────│    │──────────────────│    │─────────────────────│
│ LSTM / Prophet  │    │  Double Auction  │    │ Solidity Contract   │
│ 48h forecast    │    │  Order Book      │    │ onlyOperator guard  │
│ Confidence band │    │  Price-time      │    │ Immutable TX hash   │
│ Irradiance sim  │    │  priority match  │    │ Carbon credits      │
│                 │    │  ₹2–₹15 bounded  │    │ Sepolia Testnet     │
└─────────────────┘    └──────────────────┘    └─────────────────────┘
```

### What Makes It Different

- ✅ **Not net metering** — sellers earn market rate, not utility's throwaway buyback price
- ✅ **Not a centralized exchange** — auction mechanism is the price discovery engine
- ✅ **Bounded dynamic pricing** — `price = clamp(base + Δ(demand−supply) × 0.5, ₹2, ₹15)` — no negative prices
- ✅ **Async blockchain settlement** — trades execute instantly off-chain; Sepolia TX confirms in background
- ✅ **Carbon credit auto-tracking** — every trade records `units × 0.4 kg CO₂` offset, verifiable on-chain
- ✅ **Community battery pool** — 50 kWh virtual buffer absorbs surplus when no buyer is live

### End-to-End Flow

```
1. Household submits energy reading (produced kWh + consumed kWh)
        ↓
2. Solar service computes surplus via irradiance curve
        ↓
3. AI microservice returns 48h forecast + confidence band
        ↓
4. Order book receives surplus → creates sell order
        ↓
5. Double auction clears: lowest ask meets highest bid → trade executes
        ↓
6. Carbon credit computed: units × 0.4 kg CO₂ → credited to seller
        ↓
7. Blockchain service writes settled trade to Sepolia (async, non-blocking)
        ↓
8. WebSocket broadcasts updated prices + order book to all dashboards
```

## 🗄️ Database Schema

```sql
CREATE TABLE users (
  id              SERIAL PRIMARY KEY,
  name            TEXT,
  email           TEXT UNIQUE,
  password_hash   TEXT,              -- bcrypt, never plaintext
  wallet_address  TEXT,
  panel_kw        FLOAT DEFAULT 5,
  reputation      FLOAT DEFAULT 100
);

CREATE TABLE energy (
  id        SERIAL PRIMARY KEY,
  user_id   INT REFERENCES users(id),
  produced  FLOAT,
  consumed  FLOAT,
  surplus   FLOAT GENERATED ALWAYS AS (produced - consumed) STORED,
  timestamp TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE TABLE trades (
  id         SERIAL PRIMARY KEY,
  seller_id  INT REFERENCES users(id),
  buyer_id   INT REFERENCES users(id),
  units      FLOAT,
  price      FLOAT,
  co2_saved  FLOAT,                  -- units × 0.4 kg
  tx_hash    TEXT,
  status     TEXT DEFAULT 'pending',
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE TABLE community_pool (
  id          SERIAL PRIMARY KEY,
  stored_kwh  FLOAT DEFAULT 0,
  capacity    FLOAT DEFAULT 50,
  updated_at  TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### Bounded Pricing Engine
```javascript
const MIN_PRICE = 2;   // ₹/kWh floor
const MAX_PRICE = 15;  // ₹/kWh ceiling

function calculatePrice(supply, demand) {
  const raw = 5 + (demand - supply) * 0.5;
  return parseFloat(Math.min(MAX_PRICE, Math.max(MIN_PRICE, raw)).toFixed(2));
}
```

### Order Book Matching (Price-Time Priority)
```javascript
function matchOrders(sellOrders, buyOrders) {
  const asks = [...sellOrders].sort((a, b) => a.price - b.price || a.ts - b.ts);
  const bids = [...buyOrders].sort((a, b) => b.price - a.price || a.ts - b.ts);
  const trades = [];
  let i = 0, j = 0;
  while (i < asks.length && j < bids.length) {
    if (bids[j].price < asks[i].price) break;
    const units = Math.min(asks[i].remaining, bids[j].remaining);
    trades.push({
      seller: asks[i].userId,
      buyer:  bids[j].userId,
      units,
      price:    (asks[i].price + bids[j].price) / 2,
      co2Saved: parseFloat((units * 0.4).toFixed(3))
    });
    asks[i].remaining -= units;
    bids[j].remaining -= units;
    if (asks[i].remaining === 0) i++;
    if (bids[j].remaining === 0) j++;
  }
  return trades;
}
```

### Secured Smart Contract
```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.20;

contract EnergyTrade {
    address public operator;

    struct Trade {
        address seller;
        address buyer;
        uint    units;
        uint    price;
        uint    co2Saved;   // grams (units × 400)
        uint    timestamp;
    }

    Trade[] public trades;

    event TradeExecuted(
        address seller, address buyer,
        uint units, uint price, uint co2Saved, uint timestamp
    );

    modifier onlyOperator() {
        require(msg.sender == operator, "Not authorised");
        _;
    }

    constructor() { operator = msg.sender; }

    function recordTrade(
        address _seller, address _buyer, uint _units, uint _price
    ) external onlyOperator {
        uint co2 = _units * 400;
        trades.push(Trade(_seller, _buyer, _units, _price, co2, block.timestamp));
        emit TradeExecuted(_seller, _buyer, _units, _price, co2, block.timestamp);
    }

    function getTrades() external view returns (Trade[] memory) {
        return trades;
    }
}
```

## 🔧 Getting Started

### Prerequisites

```bash
Node.js >= 18
Python >= 3.9
PostgreSQL >= 14
MetaMask wallet (for blockchain features)
```

### 1. Clone the Repository

```bash
git clone https://github.com/BihariBabu/energygrid.git
cd energygrid
```

### 2. Environment Variables

Create `.env` in `server/`:
```env
PORT=5000
DB_URI=postgresql://user:password@localhost:5432/energygrid
JWT_SECRET=your-long-random-secret
JWT_REFRESH_SECRET=another-random-secret
AI_SERVICE_URL=http://localhost:5001
RPC_URL=https://sepolia.infura.io/v3/YOUR_INFURA_KEY
PRIVATE_KEY=your_wallet_private_key
CONTRACT_ADDRESS=deployed_contract_address
BCRYPT_ROUNDS=12
```

### 3. Backend

```bash
cd server
npm install
npm start
# runs on http://localhost:5000
```

### 4. AI Service

```bash
cd ai_service
pip install flask prophet pandas numpy scikit-learn
python app.py
# runs on http://localhost:5001
```

### 5. Frontend

```bash
cd client
npm install
npm start
# runs on http://localhost:3000
```

### 6. Smart Contract

```bash
cd contracts
npm install --save-dev hardhat @nomicfoundation/hardhat-toolbox
npx hardhat compile
npx hardhat run scripts/deploy.js --network sepolia
```
